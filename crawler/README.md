# EBS AI 데이터 수집

## 사용방법

1. node를 설치합니다. (nvm을 이용한 방법 추천)
2. 현재 폴더에서 `npm install`을 통해 패키지를 설치합니다.
3. `npm run start`를 통해 스크립트를 실행시킬 수 있습니다.
